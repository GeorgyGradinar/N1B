# Локализация (i18n)

Поддерживаются языки: **RU**, **EN**.

## Как добавить новый язык

1. Скопируйте `en.json` в новый файл с кодом языка, например `de.json` (для немецкого).
2. Переведите все строки в новом файле. Сохраните ту же структуру ключей и плейсхолдеры (`{year}`, `{years}` и т.д.).
3. В `app/composables/useI18n.ts`:
   - добавьте импорт: `import de from '~/locales/de.json'`;
   - добавьте локаль в объект `messages`: `const messages = { ru, en, de }`;
   - расширьте тип `locale`: `ref<'ru' | 'en' | 'de'>('ru')`;
   - в `setLocale` и `initLocale` добавьте проверку для новой локали;
   - в массив `availableLocales` добавьте `{ code: 'de' as const, name: 'DE' }`.

После этого переключатель языков в шапке покажет новый вариант.
